💳 "งบ" / กด "เติมเข้า" = จ่ายเงินจริงไหม? ไม่ใช่การจ่ายเงินจริงค่ะ — "งบ" คือเพดานที่เจ้านายตั้งไว้เพื่อติดตามและเตือนว่าใช้ AI ไปเท่าไหร่ (เครื่องวัด/วางแผน) · เงินจริงจ่ายที่ผู้ให้บริการโดยตรง: Claude → บัญชี Anthropic (ผ่าน API key ที่ตั้งใน Dify), Gemini → Google, SlipOK → แพ็กเกจ SlipOK · ถ้าเครดิตจริงที่ผู้ให้บริการหมด AI จะหยุดเองไม่ว่าหน้านี้จะเขียนกี่ % ก็ตาม

🤖 ทำไม Claude มี 2 ตัว (Haiku + Sonnet)? ไพลินทั้งคู่ค่ะ — Haiku (ถูก/เร็ว) อ่านว่าลูกค้าต้องการอะไร, Sonnet (ฉลาดกว่า/แพงกว่า) เขียนคำตอบจริง → ใช้ตัวถูกคัดกรองก่อน เรียกตัวแพงเฉพาะตอนตอบ = ประหยัดต้นทุน
